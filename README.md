# Pioneer Legacy for Homey

Homey SDK3 app for Pioneer Legacy AV receivers using the Pioneer Telnet protocol.

## Features

- ✅ Power on / off
- ✅ Volume up / down
- ✅ Volume in dB
- ✅ Mute
- ✅ Source selection
- ✅ Listening mode selection
- ✅ Status synchronization
- ✅ Diagnostics
- ✅ Debug command

## Supported receivers

Currently tested on:

- Pioneer SC-LX87

Other Pioneer Legacy receivers will be added in future releases.

## Installation

Install the app on your Homey Pro and configure:

- IP address
- Telnet port (8102)
- Timeout
